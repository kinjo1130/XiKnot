会社名の構成の考案

心理学系のプロダクトなので、Psycを入れたかった

案1「SaiNaut / PsyNaut（サイノート）」
・sai（psy）＝ ψ
・Naut=ギリシャ語の「nautes（船乗り）」に由来
・研究者を”航海者”として扱う世界観
案2「SaiCraft / PsyCraft（サイクラフト）」
・saiku（サイク：細工）が含まれる
・「実験を作る」という意味と重なる
・”C”で音がつながっているので響きはいいかも


- 会社名 → 船や航海
- アンケートフォーム（プロダクト1）→ 船に乗せる荷物
- 売買プラットフォーム（プロダクト2）→ 港


## 会社名 XiKnot


専門性×伴走（“プロとしての共同体”が主役）

Mission
専門性で現場の前進を支え、成果まで伴走する。

Vision
「作る人」と「使う人」が分断されず、継続的に改善が回る現場が増えている。

Value

パートナーとして動く（成果にコミット）
難所を引き受ける（専門性で突破する）
早く作って早く学ぶ（短いフィードバックループ）
ログで知恵を残す（意思決定の記録）
チームで積み上げる（協働・レビュー・共有）



## 詳細版


Mission（存在理由）

専門性で、現場の前進を支え、成果まで伴走する。

（ニュアンス違いの候補）

専門性で、成果に到達するまで伴走する。

難所を引き受け、現場を前に進める。

Vision（目指す未来）

爆速に作れる時代でも、安心して長く使えるソフトウェアが当たり前になっている。

（ニュアンス違いの候補）

「作って終わり」ではなく、改善が回り続ける現場が増えている。

使う人と作る人が分断されず、継続的に価値が積み上がる世界。

Value（行動指針）— 5つに絞った版

成果にコミットする（作ることがゴールじゃない）

難所を引き受ける（民主化されない専門領域を尖らせる）

最短で価値を出す（短いサイクルで検証し前に進める）

長く凌げる設計をする（運用・保守・移行まで見据える）

ログで積み上げる（意思決定と知見を資産化する）

会社紹介にそのまま使える一文（外向け）

「専門性で難所を突破し、成果まで伴走するソフトウェアチームです。」

「速く作り、長く使える形に育てる。成果に責任を持って伴走します。」
